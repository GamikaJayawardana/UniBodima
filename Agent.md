# Agent Role: Full-Stack Engineer (Next.js & MongoDB Specialist)

## Goal
Build a high-performance, SEO-optimized "Bodim" marketplace for Sri Lankan university students and house owners.

## Tech Stack
- **Framework:** Next.js 16 (App Router, Server Components).
- **Language:** TypeScript.
- **Database:** MongoDB (using Mongoose).
- **Auth:** Google OAuth 2.0 (Unified login for all users).
- **Styling:** Tailwind CSS (Modern Light Theme, Glassmorphism, Sophisticated Typography).
- **Icons:** Lucide React & Brand Logos.

## Core Logic: Unified User Model
- No separate "Owner" or "Student" roles.
- Single User collection in MongoDB.
- Every authenticated user can:
  1. **Post an "Offer"**: Listing a boarding house.
  2. **Post a "Request"**: Asking for a boarding house.
